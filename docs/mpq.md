---

## 📁 Was sind MPQ-Dateien?

MPQ (MoPaQ) ist ein Archivformat, das von Blizzard für Diablo 2 verwendet wird. Es enthält alle Spieldaten wie Grafiken, Sounds und Animationen.

---

## 📋 Erforderliche Dateien

| Datei | Größe | Inhalt |
|-------|-------|--------|
| `d2exp.mpq` | ~250 MB | Expansion Content |
| `d2music.mpq` | ~50 MB | Musik |
| `d2sfx.mpq` | ~100 MB | Soundeffekte |
| `d2data.mpq` | ~300 MB | Kerndaten |
| `d2char.mpq` | ~150 MB | Charaktergrafiken |
| `d2vid.mpq` | ~100 MB | Videos |

## 🔧 Konfiguration

Die MPQ-Dateien werden automatisch erkannt, wenn sie im Spielverzeichnis liegen. Alternativ können Sie den Pfad in der Konfiguration angeben:

```bash
# Environment Variable
export DIABLO2_PATH=/pfad/zu/diablo2
```
