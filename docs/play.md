# Ouroboros Collective - Spielanleitung

**ARE-Diablo2-BaalAal**

---

## 🎮 Spiel starten

### Desktop Version
1. Go Server starten: `go run .`
2. Im Browser öffnen: `http://localhost:8080`

### Mobile/Web
1. Web-Client starten: `cd web-client && npm run dev`
2. Mobile Browser öffnen: `http://localhost:3000`

---

## 🎯 Steuerung

### Desktop
| Taste | Aktion |
|-------|--------|
| WASD | Bewegung |
| SPACE | Angriff |
| E | Interagieren |
| ESC | Menü |

### Mobile
| Steuerung | Aktion |
|-----------|--------|
| Virtual Joystick | Bewegung |
| Tap auf Gegner | Angriff |
| Hotbar | Fähigkeiten |
| Swipe | Kamera |

---

## 📱 Mobile Interface

### Virtual Joystick
- Linke Bildschirmhälfte für Bewegung
- Analog-Stick für präzise Steuerung

### Hotbar
- 6 Fähigkeiten-Slots
- Tippen zum Aktivieren
- Long-press für Details

---

## 🐍 Axiomatic Engine

### Resonanz-System
Das Ouroboros Cycle System:
1. Aktion ausführen
2. Resonanz erhöhen
3. Feedback verarbeiten
4. Zustand aktualisieren

### Kappa-Space
- Koordinaten für alle Spielobjekte
- Deterministic positioning
- Multiplayer-Sync
