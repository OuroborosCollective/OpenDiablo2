# CLAUDE.md - Ouroboros Collective Web Client

**ARE-Diablo2-BaalAal - Mobile/Web Frontend**

## Projekt-Übersicht

- **Framework:** Next.js 14+
- **Sprache:** TypeScript
- **Engine:** Axiomatic BaalAal
- **Ziel:** Mobile-optimiertes Frontend

## Technologie-Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Babylon.js (3D)
- WebSocket (Echtzeit-Kommunikation)

## Entwicklung

```bash
npm install
npm run dev   # http://localhost:3000
npm run build # Production
```

## Mobile-Features

- Virtual Joystick
- Touch-Controls
- Responsive Design
- PWA Support
